n개 종류의 동전. (최대 100가지)
k원을 맞춘다.(최대 10,000원)

동전을 몇개를 쓰든 상관없다.

이때 k원을 맞출 수 있는 경우의 수는?

1. dp[i][j]를 마련 i는 동전의 종류 n j는 금액 k.
2. dp[i][j]는 i종류의 동전으로 j원을 맞출때 가능한 모든 경우의 수를 나타냄.
3. i번째 종류의 동전의 금액 Ai 와, j번째 금액의 Bj. Ai<=Bj(동전을 사용할 수 있는 경우)에
Bj%Ai == 0 , != 0 인경우를 나눈다. ==0 인 경우 Bj/Ai = k 라 할때, dp[i-1][j-k*Bj/Ai]값을 모두 더한다.
!=0 인경우, dp[i][j-Bj&Ai]값에 dp[i][Bj%Ai]값을 더한다.

dp[i][j]=dp(i,j-{1,2,3,..,j/arr[i]}arr[i])

메모리 용량이 제한적이다
