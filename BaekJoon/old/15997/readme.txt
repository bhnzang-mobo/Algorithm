모든 경우의 수를 계산할 계획.
어떤 모든 경우의 수?
3^6가지 경우. 6게임에 대해 3가지 모든 경우를 고려함.

각 경우에 대해 해당 경우가 발생할 확률을 저장.
0~727(3^6-1) 가지의 경우의 수를 3진법으로 변환. 각각의 자료를 가져와 각 사건이 발생할 확률을 저장한다.
각 사건에 대해 우승한 팀에게 확률을 더해준다.

DFS로 모든 경우의 수를 탐색할 계획.

경우의 수는 digit에 6자리 3진법 수. ex) 012021(3)
0은 1번째 게임에서 앞 팀이, 1은 비기고, 2는 뒷 팀이 승리한경우.
각 경우의 수를 모두 도합. 승 패로 점수 산출.
우승하는 팀에게 경우의 확률을 부여. 동점자가 있는 경우
동점자 수만큼 1/n 하여 확률을 더한다.
